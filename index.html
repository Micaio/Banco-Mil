Troféus

Para ganhar um troféu de ouro o jogador precisa ganhar uma partida.

Para ganhar um troféu de prata o jogador precisa termiar em segundo lugar de uma partida.

Para ganhar um troféu de bronze o jogador precisa terminar em terceiro lugar em uma partida.

Para Obter um troféu roxo o jogador terar que ganhar 5 vezes seguidas.

Para Obter um troféu preto o jogador terar que ganhar 3 vezes seguidas.
